# Toolpot

A function calling proxy for OpenAI and Anthropic API, can be used with any client without tool support, such as Jan or Continue.

## Usage

```
deno run --allow-net --allow-env --allow-read ./serve.ts
```

or

```
deno run --allow-net --allow-env --allow-read ./serve.ts --hostname 127.0.0.1 --port=9000
```

Pre-alpha version, use at your own risk!
